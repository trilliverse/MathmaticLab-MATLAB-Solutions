## üìñ Introduction

This repository contains the MATLAB source code for the laboratory assignments of the **"Mathematical Experiment"** course at South China University of Technology (SCUT). It is intended to serve as a reference for study and exchange, helping to deepen the understanding of MATLAB's applications in numerical computation, mathematical modeling, and data visualization.

## üõ†Ô∏è Requirements

* **Recommended Version**: MATLAB **R2024a** or **R2024b**.
* **Required Toolboxes**:
  - Optimization Toolbox
  - Statistics and Machine Learning Toolbox
  - Curve Fitting Toolbox

## üöÄ Usage

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/trilliverse/MathmaticLab-MATLAB-Solutions.git
   ```

2. Open and run the corresponding assignment files in MATLAB.
