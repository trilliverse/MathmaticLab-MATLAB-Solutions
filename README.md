## 📖 项目简介

本仓库包含了华南理工大学 **《数学实验》** 课程的课后实验作业 MATLAB 源代码，旨在提供学习交流参考与深入理解 MATLAB 在数值计算、数学建模及数据可视化中的应用。

## 🛠️ 环境要求

* **推荐版本**：MATLAB **R2024a** 或 **R2024b**。
* **必要工具箱**：
  - Optimization Toolbox
  - Statistics and Machine Learning Toolbox
  - Curve Fitting Toolbox


## 🚀 使用方法

1. 克隆本仓库到本地：

   ```bash
   git clone https://github.com/trilliverse/MathmaticLab-MATLAB-Solutions.git
   ```

2. 使用 MATLAB 打开并运行对应的作业文件。

## 📂 目录索引

| **章节**   | **实验主题**     | **核心内容 / 算法**                       |
| ---------- | ---------------- | ----------------------------------------- |
| **Lab 01** | MATLAB 基础      | 矩阵运算、基本绘图、逻辑控制流            |
| **Lab 02** | 数据拟合         | 线性/非线性拟合、最小二乘法原理           |
| **Lab 03** | 微分方程         | ODE 求解器应用、数值模拟分析              |
| **Lab 04** | 插值方法         | Lagrange 插值、分段线性、三次样条         |
| **Lab 05** | 特征值与特征向量 | 矩阵特征值提取、实际应用案例分析          |
| **Lab 06** | 古典概型         | 随机事件模拟、蒙特卡洛 (Monte Carlo) 方法 |
| **Lab 07** | 回归分析         | 多元线性回归、统计推断与显著性检验        |
